Proof of concept for the server API and data model.

The idea is to remove all complexity and implement a simple number guessing game, having all elements of the actual battleship game however.

"Prepare" - Player enters a number from 1-100. He submits the number to the server, where to same processing takes place as in the actual game (including validation), except the payload is not an array of ships but a plain number.

"Match" - A list of ready players is displayed and two players can form a pair of oppenents. Almost the same as in the actual game from a data perspective.

"Battle" - Guessing instead of shooting
